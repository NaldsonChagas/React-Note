# React Note

This is a web application made with Node.Js and React. With this app, you can add notes for reminders


**Functions and Pages:**

* Home Page (Login and Register
![Home Page](https://i.imgur.com/mRnTXkX.png)

* List, Save and Update notes

* Update user info

Backend:

* express
* express-kun
* jsonwebtoken
* sequelize

Frontend:
* axios
* bootstrap
